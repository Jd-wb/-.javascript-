JavaScript for Beginners

1. Introduction to JavaScript

What is JavaScript?

JavaScript is a versatile programming language that enables you to create interactive and dynamic web content. It is essential for web development, enabling you to enhance user experience on websites.

Why Learn JavaScript?

JavaScript allows you to:

  • Add interactivity to websites (e.g., animations, forms).
  • Manipulate web content and respond to user actions.
  • Build full-fledged applications (both front-end and back-end).

Setting Up Your Environment

To get started with JavaScript, you don't need much. Just a web browser and a text editor (like VS Code or Notepad) will do!

2. JavaScript Basics

Variables

Variables store data values. You can declare variables using var, let, and const.

let name = "Alice"; // A variable that can change
const age = 30;     // A constant that cannot change

Data Types

JavaScript has several data types, including:

let number = 42;                // Number
let greeting = "Hello, World!"; // String
let isActive = true;            // Boolean
let fruits = ["apple", "banana"]; // Array
let person = { name: "Alice", age: 30 }; // Object

Operators

Operators are used to perform operations on variables and values. Common operators include:

  • Arithmetic Operators: +, -, *, /
  • Comparison Operators: ==, ===, !=, !==, >, <
  • Logical Operators: && (AND), || (OR), ! (NOT)

Functions

Functions are reusable blocks of code. You can define and call them:

function greet() {
  console.log("Hello, World!");
}
greet(); // Calls the function and outputs: Hello, World!

Conditionals

Conditionals allow you to run code based on certain conditions:

let age = 20;
if (age >= 18) {
  console.log("You are an adult.");
} else {
  console.log("You are a minor.");
}

Loops

Loops let you repeat a block of code:

// for loop
for (let i = 0; i < 5; i++) {
  console.log(i); // Outputs: 0, 1, 2, 3, 4
}

3. Arrays and Array Methods

Arrays are used to store multiple values in a single variable. JavaScript provides many methods to work with arrays.

let fruits = ["apple", "banana", "cherry"]; // An array of fruits
console.log(fruits[0]); // Accessing the first element: "apple"

fruits.push("orange"); // Adding a new fruit to the end
console.log(fruits); // Outputs: ["apple", "banana", "cherry", "orange"]

fruits.pop(); // Removing the last fruit
console.log(fruits); // Outputs: ["apple", "banana", "cherry"]

Common Array Methods:

  • push(): Adds an item to the end of the array.
  • pop(): Removes the last item from the array.
  • shift(): Removes the first item from the array.
  • unshift(): Adds an item to the beginning of the array.
  • splice(): Adds or removes items from the array at a specific index.
  • forEach(): Executes a function for each element in the array.

4. Objects and Object Methods

Objects are collections of key-value pairs. They allow you to group related data and functions.

let person = {
    name: "Alice",
    age: 30,
    greet: function() {
        console.log("Hello, " + this.name);
    }
};
person.greet(); // Outputs: Hello, Alice

Common Object Methods:

  • Object.keys(): Returns an array of a given object's property names.
  • Object.values(): Returns an array of a given object's property values.
  • Object.entries(): Returns an array of a given object's key-value pairs.

5. JSON (JavaScript Object Notation)

JSON is a lightweight data interchange format. It's easy to read and write for humans and easy to parse and generate for machines.

// JavaScript object
let person = { "name": "Alice", "age": 30 };

// Convert to JSON string
let jsonString = JSON.stringify(person);
console.log(jsonString); // Outputs: {"name":"Alice","}
